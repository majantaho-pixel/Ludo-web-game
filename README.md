# Ludo Web Game

**Developer:** Zeeshan Rao

## Overview
This is a web-based Ludo game. It includes all the core features of a classic Ludo game including:

- Fully interactive Ludo board
- 4 player tokens with color-coded pieces
- Dice roll functionality
- Turn-based gameplay
- Winning and game-over conditions
- Responsive design for desktop and mobile

## Features
- Drag and drop player tokens to move on the board
- Dice rolling animation
- Player turns with proper validation
- Customizable token names and colors
- Score tracking

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- etc.
